ChimeraLLM

ChimeraLLM sends your natural-language requests to a configured LLM and executes the resulting ChimeraX commands. Choose Settings to use either an OpenAI-compatible API (API key and endpoint) or GitHub Copilot (sign in with the device flow).

Use Cancel to stop a long-running request. Clear removes the on-screen transcript and the in-memory conversation context.

Command: chimerallm opens this panel; you may add an optional prompt.
